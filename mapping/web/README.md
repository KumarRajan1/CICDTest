# Mappls Web Plugins

Mappls Web Plugins is a JavaScript library for interactive maps provided by Mappls. This library is compatible with both our Web SDKs — Vector Maps and our legacy Raster Maps.

MAPPLS will be there in every step of the way, building new tools that help you navigate, explore, and get things done — wherever you are. Users can touch or interact with map features clearly overlaid on their view of the world.

Our APIs, SDKs, and live updating map data available for [200+ nations and territories](https://github.com/MapmyIndia/mapmyindia-rest-api/blob/master/docs/countryISO.md) give developer-friendly, easy-to-integrate plugins to add capabilities like intelligent search and routing to your web or mobile applications.

➡️ Get your API key here: [https://apis.mappls.com/console/](https://apis.mappls.com/console/)

- Indic Support (Hindi)

### [Directions Plugin](https://github.com/mappls-api/mappls-web-plugins/tree/main/directions-plugin)

[Read more](https://about.mappls.com/api/web-sdk/vector-plugin-example/Direction/mappls-direction-plugin)
