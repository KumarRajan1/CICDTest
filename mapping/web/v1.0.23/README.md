# Mappls Annotation Plugin

The **Mappls Annotation Plugin** simplifies how to set and adjust the visual properties of annotations on a Mappls map.

This plugin allows you to plot:
- **Symbols** (markers)
- **Lines**
- **Fills** (polygons)


---

##  Add the Dependency

Add the following dependency in your **app-level `build.gradle`** file:

```groovy
implementation 'com.mappls.sdk:annotation-plugin:1.0.1'
